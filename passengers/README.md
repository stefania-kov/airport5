# Информация для пассажиров

Этот раздел поможет вам сориентироваться в аэропорту.

## **Важные шаги** перед вылетом:

1.  Заблаговременно пройдите онлайн-регистрацию на сайте вашей авиакомпании.
2.  Уточните нормы провоза багажа и ручной клади.
3.  Прибывайте в аэропорт за **3 часа** до вылета для международных рейсов и за **2 часа** для внутренних.
